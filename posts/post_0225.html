📚 Grammar Points

～てみる (N4) - Trying Something

Used when trying something out to see how it goes.

行ってみよう！ → Let's try going!

～みたい (N3) - Appears like / Seems like

Indicates that something appears to be a certain way based on what the speaker sees or perceives.

リビングルームみたいな感じ → It seems like a living room.

～たり～たりする (N4) - Listing Multiple Actions

Used to list multiple actions or states in a non-exhaustive way.

電子レンジだったり、お茶のセットが置いてあります。 → There are things like a microwave and a tea set.

～ながら (N3) - While Doing Something

Indicates two actions happening at the same time.

寝ながら充電も可能です。 → You can charge while sleeping.

📖 Vocabulary List

近代的（きんだいてき）- Modern (N3)

Example: めっちゃ近代的。 → It's really modern.

自炊（じすい）- Cooking for oneself (N3)

Example: 自炊もできるみたい。 → It seems like you can cook for yourself.

お手洗い（おてあらい）- Bathroom / Toilet (N4)

Example: こちらがお手洗いとお風呂もございます。 → Here is the bathroom and the bath.

切り替える（きりかえる）- To Switch / Change (N3)

Example: 水圧、切り替えられるタイプ。 → A type where the water pressure can be switched.

📌 Practice Questions

  1. 「～てみる」を使って、自分の部屋で試したことを1つ書いてみよう。
  2. 「～ながら」を使って、同時にできることを1つ書いてみよう。
  3. 「～たり～たりする」を使って、週末にすることを1つ書いてみよう。
